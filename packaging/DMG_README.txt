Truck Packer — install from this disk image
============================================

1. Quit your DAW.

2. Drag "Truck Packer.vst3" into:
     ~/Library/Audio/Plug-Ins/VST3/
   (In Finder: Go menu → Go to Folder… → paste that path.)

3. Drag "Truck Packer.component" into:
     ~/Library/Audio/Plug-Ins/Components/

4. Open your DAW and rescan plug-ins if needed.

For a silent system-wide install (admin), use the .pkg from the same release instead of this DMG.
